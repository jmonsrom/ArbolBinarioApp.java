Implementación de Árbol Binario en Java

Este proyecto pretende hacer una implementación básica de un Árbol Binario de Búsqueda, por sus siglas en inglés (BST).

¿Qué es un Árbol Binario?
Es una estructura de datos jerárquica donde cada nodo tiene, como máximo, dos hijos (izquierdo y derecho). En un árbol de búsqueda, los valores menores van a la izquierda y los mayores a la derecha.

Funcionamiento del Recorrido Inorden
El recorrido **Inorden** visita los nodos en este orden: **Izquierda -> Raíz -> Derecha**. Esto tiene la propiedad única de mostrar los elementos del árbol en orden ascendente.

Cómo ejecutar el programa
1. Clona el repositorio.
2. Compila con `javac ArbolBinarioApp.java`.
3. Ejecuta con `java ArbolBinarioApp`.
